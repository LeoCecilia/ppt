skew

如何让内容不倾斜!

到了公布答案的时刻了

						

Click


.button {
	transform:skewX(-45deg);
}
.button > div {
	transform:skewX(45deg);
}

但是这做法很明显不符合语义化的需求

而且这也破坏了文本结构的纯净度

思路

把样式应用到伪元素当中

再对伪元素进行变形

如何让伪元素自动继承宿主元素的尺寸呢

					
.button{
	position: relative;
}
.button::before{
	content: '';
	position: absolute;
	top: 0; right: 0;
	bottom:0; left: 0;
	
}						
					
				
					
.button{
	position: relative;
}
.button::before{
	content: '';
	position: absolute;
	top: 0; right: 0;
	bottom:0; left: 0;
	z-index:-1;
	background: #58a;
	transform:skew(45deg);
}

总结

利用伪元素以及定位属性产生一个block

为其设置样式

并将其放在宿主元素的下层

拓展延伸

  • 绘制梯形
